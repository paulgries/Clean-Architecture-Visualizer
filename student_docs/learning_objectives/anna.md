Learning objectives

1. Advance responsive front-end development using modern frameworks

Develop the ability to design and implement responsive, accessible user interfaces using React and TypeScript, building on existing HTML/CSS/JavaScript knowledge.

2. Strengthen CSS and visual design decision-making

Improve proficiency in CSS layout, styling, and design principles (e.g., spacing, hierarchy, responsiveness) in order to create visually clear and usable interfaces. 

3. Learn to design and implement automated GUI tests

Learn to design and write automated GUI tests using an industry-relevant testing framework, applying them to verify the correctness and stability of the group’s front-end code.

4. Use Figma more efficiently for interactive design and iteration

Expand knowledge of Figma’s advanced features (such as components, auto-layout, and interactive prototyping) to create and iterate on interface designs more efficiently and communicate design decisions.

5: Efficient meeting preparation and participation

Develop effective meeting preparation and participation skills by consistently preparing agendas, clarifying discussion goals, and documenting decisions and action items to support efficient and productive team meetings.

6: Sustainable time management across courses

Improve personal time management and prioritization skills by intentionally scheduling project work alongside other course commitments to ensure steady progress without allowing the project to consume a disproportionate share of time.
